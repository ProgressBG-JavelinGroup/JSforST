The Object object

In JavaScript, objects can be seen as a collection of properties. Or as a mapping between keys and values.
Property names (the keys) can be either a String or a Symbol value. If they are not, JavaScript tries to typecast them.
Property values (the values) can be values of any type, including other objects
JavaScript objects can be used to implement associative array/dictionary/hashmaps data structures.

			var point = {
				x: 10,
				y: 10,
				draw: function(){
					// do something to draw the point
				}
			}
			console.log(typeof point); // object
In JavaScript methods are object properties which value is a function!

Accessing object properties

Dot notation
Square bracket notation

The Dot notation


			object.property

Property must be a valid identifier!


			var point = {};

			// set object property:
			point.x = 10;

			// gat object property:
			var x = point.x;

			// example for invalid property name
			point.2; // SyntaxError: Unexpected number

The Square brackets notation


			object['property']

The string does not have to be a valid identifier; it can have any value, like "42", or even " " (a space).


			var point = {};

			// set object property:
			point['x'] = 10;

			var propName = 'x';

			// gat object property:
			var x = point[propName];

			// next is also valid
			point[2] = 10;

			console.log(point);

Every JavaScript object is created by the Object build in object and inherits its properties and methods.

Objects will be discussed deeply in next sections

The Array object

Arrays are regular objects, which keys are integers, and for which the 'length' property is defined.
Neither the length of a JavaScript array nor the types of its elements are fixed.
Arrays cannot use strings as element indexes
JavaScript arrays are zero-indexed

			var colors = ['red', 'green', 'blue'];
			console.log(typeof colors); // object
		

			// Array values can be of any type:
			var userData = [ 24, 'Bulgaria', {avatar:'avatar245.png'}];

Accessing array elements

The first element of an array is at index 0
The last element is at the index: arr.length-1

			var colors = ['red', 'green', 'blue'];
			console.log( colors[1] ); // 'green'

			// add new element:
			colors[3] = 'black';
			console.log(colors);
			// (4) ["red", "green", "blue", "black"]
Arrays in JavaScript are created by the build in Array object and inherits its properties and methods.

The Date object

Dates/times in JavaScript are represented by the Date build in object
Date objects are based on a time value that is the number of milliseconds since 1 January, 1970 UTC.
Unlike other JavaScript object types, JavaScript Date objects have no literal syntax.
I.e. you always have to create a new Date object by the constructor

			var now = new Date();
			console.log(`now is: ${now}`);

All Date instances inherit from Date.prototype


			var now = new Date();
			console.log( now.getFullYear() );
